📁 File Naming Conventions

  • Enrollment JSON: {CN}_enrollment_{timestamp}.json
  • Configuration: {CN}_enrollment_{ID}_{timestamp}.json
  • CSR Files: {CN}_enrollment_{ID}_{Algorithm}.csr
  • Certificates: {CN}_{Algorithm}_certificate.pem
  • Config Symlink: {CN}.json → Latest config

🔄 Key Features

  • No Duplicate Enrollments: Renewals update existing enrollment
  • Configuration Management: All settings saved for easy renewal
  • Enrollment Discovery: Find and reuse existing enrollments
  • Automatic Polling: Waits for CSR, warnings, and deployment
  • Safe File Naming: Timestamps prevent overwrites
  • Multiple Certificate Support: Manage many certificates easily
  • Cron Compatible: Can be automated for renewals
